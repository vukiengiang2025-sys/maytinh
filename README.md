# 📱 SmartyCalc - Máy Tính Cầm Tay Dễ Thương

SmartyCalc là một ứng dụng máy tính được thiết kế dành cho học sinh từ Lớp 1 đến Lớp 12 với giao diện thân thiện, màu sắc pastel và hiệu ứng mượt mà.

## ✨ Tính năng
- **Bản Cơ Bản (Lớp 1-5):** Cộng, trừ, nhân, chia, phần trăm.
- **Bản Nâng Cao (Lớp 6-12):** Lũy thừa, căn bậc hai, lượng giác (sin, cos, tan), logarit, số pi, số e, giai thừa.
- **Nhật ký:** Lưu lại lịch sử các phép tính đã thực hiện.
- **Giao diện:** Cute, dễ thương, tối ưu cho cả web và di động.

## 🚀 Cách Build APK từ GitHub
Để có được file APK, bạn hãy làm theo các bước sau:
1. Pous code này lên kho lưu trữ (repository) của bạn trên GitHub.
2. Vào tab **Actions** trên GitHub repository của bạn.
3. Chọn workflow **Build APK**.
4. GitHub sẽ tự động build và cung cấp cho bạn file `.apk` trong phần **Artifacts** của mỗi lần build thành công.

## 🛠 Công nghệ sử dụng
- **React + Vite + TypeScript**
- **Tailwind CSS + Framer Motion (motion/react)**
- **Mathjs:** Thư viện tính toán mạnh mẽ.
- **Capacitor:** Công cụ chuyển đổi từ Web sang ứng dụng Android bản địa.
- **GitHub Actions:** Tự động hóa quá trình đóng gói APK.
